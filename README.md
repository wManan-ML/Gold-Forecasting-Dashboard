# Gold Forecasting Dashboard

A Streamlit-based dashboard for forecasting gold prices using Monte Carlo simulation and ARIMA benchmarking on historical GLD market data.

## 🚀 Live Demo

Access the deployed application here:

[**https://gold4cast.streamlit.app**](https://gld4cast.streamlit.app/)

## Overview

This project explores probabilistic forecasting techniques for gold price prediction using historical GLD ETF market data. The dashboard combines Monte Carlo simulation, historical market analysis, and time-series forecasting to estimate future price movements while accounting for uncertainty.

Rather than producing a single deterministic prediction, the model generates thousands of possible future price paths, allowing users to explore expected outcomes, confidence intervals, and forecast risk.

## Features

* Interactive Streamlit dashboard
* Historical GLD price visualization
* Monte Carlo simulation-based forecasting
* Forecast confidence interval estimation
* Multi-year backtesting framework
* ARIMA benchmark comparison
* Probabilistic analysis of future price scenarios

## Methodology

### Data Collection

Historical GLD ETF data is retrieved using Yahoo Finance and processed for analysis.

### Market Statistics

The system calculates:

* Daily returns
* Average return
* Historical volatility

These statistics form the foundation of the forecasting model.

### Monte Carlo Simulation

Monte Carlo simulation is the core forecasting engine of this project.

Using historical return and volatility estimates, the model generates thousands of potential future price paths. The resulting distribution is used to estimate:

* Expected future price
* Median forecast
* Confidence intervals
* Probability-based outcomes

### ARIMA Benchmark

To evaluate forecasting performance, simulation results are compared against an ARIMA time-series forecasting model.

## Results

### Backtesting Performance

* Average Forecasting Error: **10.84%**

* Multi-year historical backtesting performed

Clone the repository and install dependencies:

```bash

cd Gold-Forecasting-Dashboard

pip install -r requirements.txt
```

Launch the dashboard:

```bash
streamlit run app.py
```
